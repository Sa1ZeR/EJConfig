## Install

[![](https://jitpack.io/v/Sa1ZeR/ejconfig.svg)](https://jitpack.io/#Sa1ZeR/ejconfig)

Just add dependency

```groovy
    allprojects {
        repositories {
            // ...
            maven { url 'https://jitpack.io' }
        }
    }

    dependencies {
        implementation 'com.github.Sa1ZeR:ejconfig:Tag:$version'
    }
```
